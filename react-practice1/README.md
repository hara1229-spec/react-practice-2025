仕事・プライベートなどカテゴリーごとにタスクをグループ化する機能（後からグループは変更可能、色分けして一目で分かるようにしたい）

タスクに期限を設定可能にする。期限が近くなったらアラートを出す。何分前、何日前などいつ出すか、何回出すかもカスタマイズ可能。

タスク毎に重要度を設定、★１～３をつける。★別にソートや絞り込み可能。

他のユーザーと自分のタスクを共有する機能（編集権限の付与も可能）
